================================
This is the 0-add_integer module
================================

This module defines a function, ``add_integer()``, computes the sum of two
numbers.

Usage:
------

``add_integer()`` returns the addition of its two arguments, ``a`` and ``b``.

Import the function from its module:
    >>>add_integer = __import__("0-add_integer"").__doc__

    Examples:
    ---------

::

    Working with positive integers.

    >>>add_integer(2, 3)
    5
::
